# Trader.pro Backend

> The Trader.pro Backend is a high-performance API server built with Go, designed to power the Trader.pro trading simulation platform. It handles user authentication, trade simulation, and data persistence, integrating with PostgreSQL and external APIs like ICICI Breeze.

## Features

- User authentication via Firebase OAuth
- Simulated trading with historical market data
- RESTful API for trade management and user data
- PostgreSQL storage for users, trades, and positions
- Lightweight and fast with GoFiber

## Tech Stack

- **Language**: Go 1.21
- **Framework**: GoFiber
- **Database**: PostgreSQL
- **Authentication**: Firebase OAuth
- **Deployment**: AWS EC2 with GitHub Actions CI/CD
